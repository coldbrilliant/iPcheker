## ✨ Features

* 🌍 Lookup any IPv4 / IPv6 address
* 📡 Detailed information: Country, Region, City, ISP, Organization, ASN, Timezone, Current time, Coordinates
* 📋 Paste IP from clipboard
* 📎 Copy results instantly
* 🧪 Terminal-style UI

## 🧰 Requirements

* Python 3.x

## ⚙️ Installation & Run

ip_lookup.py
python ip_lookup.py
or
py ip_lookup.py

---

## 🕹️ Usage

1. Enter or paste IP
2. Click LOOKUP
3. View result
4. Click COPY
